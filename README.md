# Plant Watering Scheduler

**Purpose:**  
An application for managing plant watering schedules with a feature for buying and selling second-hand plants.

**Framework:**  
Flutter

**Backend:**  
Supabase

**State Management Pattern:**  
Riverpod, Provider
